                                                                                         git bash 2

команда которая показывает содержимое папки
ls 

для того что бы создать файл и сразу добавить в него строку 
echo "new line" >> new.txt

так же создать файл с любым расширением (pdf.exe.doc(docx).txt и т.д )
touch newfile.txt

так же если вы внесли несколько изменений в разных файлах 
их можно сохранить разом через команду git add newfile.txt file6.txt gigi.img и через пробел перечисляете файлы 

что бы добавить все файлы в удаленный репозиторий нужно подставить -a
git commit -a -m "coment"

если нам нужно откатится на один шаг назад 
git reset HEAD~1 

если вы хотите откатить недавнее изменение в файле 
git checkout newfile.txt 

eсли вы хотите откатить недавние изменения во всех файлах то просто ставите точку 
git checkout .

так же изменения можно помещать во временное хранилище 
пример: 
echo "new line" >> newfile.txt
у вас в файле добавится новая строка и при помощи команды git stash мы помещаем ее во временное хранилище
после помещения в хранилище она у нас не будет отображаться в файле 
но мы всегда можем достать эту строку при помощи команды git stash pop
и для того что бы удалить эту строку из временного хранилища используем команду git stash clear 

git stash сохранит ваши изменения и очистит рабочий каталог
git stash list покажет список всех сохраненных изменений
git stash apply восстановит последние сохраненные изменения обратно в рабочий каталог
git stash drop удалит последние сохраненные изменения из списка stash
